# NexBuild

AI-Native Product Decision Engine - Figure out what to build next with evidence-based feature recommendations.

## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or pnpm

### Installation

\`\`\`bash
npm install
\`\`\`

### Development

\`\`\`bash
npm run dev
\`\`\`

### Build

\`\`\`bash
npm run build
\`\`\`

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS v4
- React Router v7
- Radix UI Components
- Recharts
- Motion (Framer Motion)
